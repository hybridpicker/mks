Richtlinien für Veranstaltungsbilder

  • Erforderlich: Jede Veranstaltung muss ein Bild haben
  • Optimale Größe: 800 × 600 Pixel
  • Format: JPG oder WEBP (empfohlen)
  • Maximale Dateigröße: 2MB
  • Seitenverhältnis: 4:3 (Querformat)
  • Auflösung: 72 dpi ist ausreichend

Bilder werden auf der Startseite und in der Veranstaltungsübersicht angezeigt. Die optimale Darstellung wird mit einem Bild im Querformat mit guter Auflösung erreicht.

Tipp: Komprimieren Sie Ihre Bilder vor dem Hochladen mit Tools wie TinyPNG oder Squoosh.
